- Составьте запрос, который извлекает все записи из таблицы users по 
следующим правилам:

  - Пользователи должны быть рождены позже 23 октября 1999 года. 
  Поле `birthday`.
  - Выборка отсортирована в алфавитном порядке по полю `first_name`
  - Нужно извлечь только три записи

***solution***

SELECT * FROM users
WHERE birthday > '1999-10-23'
ORDER BY first_name
LIMIT 3
